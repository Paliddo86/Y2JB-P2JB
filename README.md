# Y2JB-P2JB

Y2JB-P2JB is an advanced JavaScript port of the original `P2JB.lua` project. It focuses on accurately translating complex Lua logic into a modern JavaScript runtime while improving structure, synchronization, and browser compatibility.

The project is currently around **92% complete**, with most core systems already converted and logically corrected for JavaScript execution.

---

## 🚀 Overview

This project recreates the original Lua-based logic in a JavaScript environment with a focus on:

- Accurate logic conversion from Lua to JavaScript
- Browser-compatible execution model
- Improved synchronization handling
- Modular and maintainable architecture
- Debug-friendly runtime structure

---

## ⚙️ Features

- Nearly full conversion of `P2JB.lua` core logic
- JavaScript-based runtime adaptation
- Worker and async execution handling
- Firmware-aware logic structure (ported behavior)
- Improved memory and state handling abstraction
- Enhanced debugging and logging system
- Cleaner and more maintainable codebase
